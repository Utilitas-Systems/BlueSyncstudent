# Tauri Icons

This directory should contain the following icon files for Tauri bundling:

- `32x32.png` - 32x32 PNG icon
- `128x128.png` - 128x128 PNG icon  
- `128x128@2x.png` - 256x256 PNG icon (for Retina displays)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

## Generating Icons

You can generate all required icons from a single source image using the Tauri CLI:

```bash
npm run tauri icon path/to/your-icon.png
```

This command will automatically generate all the required icon formats and sizes from your source image.

Alternatively, you can use online tools or image editing software to create the icons manually.
